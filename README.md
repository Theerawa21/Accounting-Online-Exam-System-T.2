# แบบทดสอบออนไลน์ การบัญชีเบื้องต้น

ระบบสอบ 40 ข้อ เวลา 40 นาที สุ่มข้อสอบ 1 จาก 3 ชุด ตรวจคะแนนอัตโนมัติ และบันทึกผลลง Google Sheet รองรับการใช้งานบนโทรศัพท์ แท็บเล็ต และคอมพิวเตอร์

หน้าเข้าสอบรองรับนักเรียน ม.4/3 จำนวน 39 คนจากไฟล์ `20260722134722.xlsx` นักเรียนกรอกเฉพาะรหัสประจำตัวนักเรียน 5 หลัก แล้วระบบจะค้นหาชื่อ–นามสกุลและห้องเรียนจากแท็บ `นักเรียน ม.4/3` ใน Google Sheet โดยไม่นำรายชื่อ เลขบัตรประชาชน หรือวันเกิดไปฝังในรีโพสาธารณะ

Google Sheet ที่เชื่อมไว้: `1OQDzQ9FiMKTtI6v7wQpEWusjf2gJG4DEWV6aqsJuKNs` (แท็บ `ชีต1`)

## ไฟล์สำคัญ

- `index.html` หน้าเว็บหลัก
- `style.css` รูปแบบหน้าจอเดสก์ท็อปและมือถือ
- `app.js` ข้อสอบ การจับเวลา ตรวจคะแนน และเชื่อมต่อ API
- `Code.gs` Backend สำหรับบันทึกและอ่านผลจาก Google Sheet

## วิธีเปิดใช้งานจริง

1. เปิด Google Sheet เป้าหมาย แล้วเลือก **ส่วนขยาย > Apps Script**
2. ลบโค้ดตัวอย่างในไฟล์ `Code.gs` ของ Apps Script แล้ววางโค้ดจากไฟล์ `Code.gs` ในโฟลเดอร์นี้
3. เปิด **การตั้งค่าโปรเจ็กต์ > พร็อพเพอร์ตี้ของสคริปต์** แล้วเพิ่มชื่อ `TEACHER_PIN` พร้อมรหัสที่ต้องการใช้สำหรับครู
4. กด **ทำให้ใช้งานได้ > การทำให้ใช้งานได้รายการใหม่ > เว็บแอป**
5. ตั้งค่า **ดำเนินการในฐานะ: ฉัน** และ **ผู้ที่มีสิทธิ์เข้าถึง: ทุกคน** จากนั้นกดทำให้ใช้งานได้
6. คัดลอก URL เว็บแอปที่ลงท้ายด้วย `/exec`
7. เปิด `app.js` แล้ววาง URL ใน `CONFIG.appsScriptUrl`
8. อัปโหลด `index.html`, `style.css` และ `app.js` ไปยังโฮสต์เว็บ เช่น GitHub Pages หรือ Netlify

## เผยแพร่ด้วย GitHub Pages

รีโพนี้มี workflow `.github/workflows/pages.yml` สำหรับเผยแพร่หน้าเว็บอัตโนมัติเมื่อมีการ push เข้า branch `main`

1. เปิด **Settings > Pages** ของรีโพ
2. ที่ **Build and deployment > Source** เลือก **GitHub Actions**
3. เปิดแท็บ **Actions** และรอ workflow `Deploy GitHub Pages` ทำงานสำเร็จ
4. เข้าใช้งานที่ `https://theerawa21.github.io/Accounting-Online-Exam-System-T.2/`

หากยังไม่ได้ใส่ Apps Script Web app URL หน้าเว็บและการทำข้อสอบจะใช้งานได้ แต่ผลจะถูกสำรองไว้เฉพาะในเบราว์เซอร์ของเครื่องนั้น

เมื่อมีการส่งข้อสอบครั้งแรก `Code.gs` จะสร้างหัวตาราง จัดรูปแบบ และตรึงแถวหัวข้อในแท็บ `ชีต1` ให้อัตโนมัติ รหัสหน้าครูจะอ่านจาก Script Property `TEACHER_PIN` และไม่ถูกเก็บไว้ในรีโพ

## การป้องกันข้อมูลสูญหาย

- ระหว่างทำข้อสอบ ระบบเก็บสถานะไว้ใน `sessionStorage`
- เมื่อส่งคำตอบ ระบบเก็บสำรองผลไว้ใน `localStorage` ก่อนส่งไป Google Sheet
- หากอินเทอร์เน็ตขัดข้อง หน้าผลจะแจ้งให้นักเรียนติดต่อครู และผลสำรองยังอยู่ในอุปกรณ์นั้น
- เมื่อใช้รหัสนักเรียนเดิมอีกครั้ง ระบบจะอัปเดตแถวเดิมในชีตเพื่อไม่ให้ข้อมูลซ้ำ

## ทดสอบในเครื่อง

หลังตั้งค่า Web app URL แล้ว ใช้รหัสนักเรียนที่มีอยู่ในแท็บ `นักเรียน ม.4/3` เพื่อตรวจการแสดงข้อมูลอัตโนมัติ หากยังไม่ได้ตั้งค่า URL ระบบจะไม่เปิดเผยหรือค้นหารายชื่อนักเรียนจากไฟล์เว็บ
